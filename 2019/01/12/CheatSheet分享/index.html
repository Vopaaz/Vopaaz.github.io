概统 Cheat Sheet 说明

前半部分公式完全手打，大概第 9 周开始接触了神器 Mathpix，后面就很多是用它识别，有错再调整的。
但是总之都是 LaTeX 公式代码，没有任何一张截图。直接编译渲染可以绝对保证打印的清晰度。

还有一个很重要的是给 R code 做了 syntax highlighting , 这个应该 95% 拿 Word 或者手写的旁友都做得没我好吧233。看代码没有高亮怎么行呢！

另外根据自己的使用习惯简单封装了一些命令，同样是方便修改。因为对 LaTeX 没怎么研究所以代码可能很丑不要在意【逃：

使用 minted 宏包做 R 语言 highlighting，需要一些额外操作。

  1. 安装 Pygments 【我不知道没有装 Python 要怎么办，但是看朋友圈里什么“Python 速成训练营”大家转的还挺勤快的应该不成什么问题【逃x
  2. LaTeX的编译命令需要添加 “--shell-escape”

最后，存在一个特别大的缺陷，是在后面章节出现了 这种蛋疼符号之后，由于 Mathpix 识别经常失败，很多点都是手动加的，加的位置和格式有时候不同。当然几乎完全不影响阅读和使用，只是希望有【闲着没事干的】学弟学妹能全整理好。
