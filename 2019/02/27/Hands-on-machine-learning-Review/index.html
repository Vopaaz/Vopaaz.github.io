理论

“范数”的意义

包含 n 个元素的向量 v 的 k 范数定义为

其中一范数对应 MAE, 二范数对应 RMSE.

0 范数只给出向量中元素的数量，而无穷大范数给出向量中元素的最大绝对值。

范数的指数越高，就越关注大的价值，而忽视小的价值。因此范数越大，对异常值越敏感。
