正式使用

以下文字中斜体的部分是暂时还没有学到过或者不懂的地方，后期需要将其外链链接到更详细的说明，或者页尾添加注释。

基概

  • 首先使用一个 <div class="container-fluid"> 包含整个 body 中的所有 html 元素
  • 让图片大小随着页面变化，只要给图片的 class 加上一个 img-responsive 即可
  • 让文字居中，给其 class 加上 text-center

按钮

  • 使用 Bootstrap 的按钮风格，给 button 的 class 属性加上 btn
    • 以下附加的按钮风格，都需要在 class 中保留原先的 btn
    • 让按钮转变为块级元素，即让其铺满整个一行，class 添加一个 btn-block
    • 让按钮采用应用的主要颜色，class 添加一个 btn-primary
    • 另一些可用的改变按钮颜色的 class 包括：
      • btn-info
      • btn-danger

网格布局

使用网格布局可以实现将多个元素放在一行，并且指定他们之间的相对宽度，同时让其可以根据显示大小自动变化。

网格布局的内容必须被放置在 <div class="container"> 或者 <div class="container-fluid"> 中。网格布局默认将页面分成 12 列，一个元素可以占据任意 1-12 列的宽度。

cell 可以被指定用于适配不同的显示器大小，分别用不同的 class 名前缀来表示：

Extra small Small Medium Large Extra Large
Prefix .col- .col-sm- .col-md- .col-lg- .col-xl-
Screen Pixel <576px ≥576px ≥768px ≥992px ≥1200px

这里 .col-**-N 中的 N 代表这个 cell 占据了页面 12 列中的 N 列，用于控制元素宽度。

布局在同一行中的元素，用同一个 <div class="row"> 标签包含在内。这样做之后，同一行的元素就可以均匀地在页面上展开，而不需要知道屏幕的分辨率有多宽。但如果同一行中安排的元素超过了 12 列，最后一个/多个元素就会被安排到下一行中。
