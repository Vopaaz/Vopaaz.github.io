虽然现在前端是框架时代了，但是对于 jQuery 还是有一定的了解比较好。因此这部分课程不跳过。

基概

所有的 jQuery 方法都是 $ 开头的，后面跟着一个 css 选择器，例如 $("button") 会选中页面上所有的 button 标签，$("button").addClass("animated bounce"); 则会给所有 button 标签添加上 animated bounce 类。

同理，用 $(".SOMECLASS") 和 $("#SOMEID") 来通过 class 和 id 对页面元素进行选择。

常用函数

  • $().addClass() 为元素添加类
  • $().removeClass() 为元素删除类
  • $().css() 根据参数的不同，获取首个匹配元素的 css 样式/为所有匹配元素修改 css 一个/多个样式，参考菜鸟教程
  • $().prop() 获取/设置 DOM 元素的属性，与 css 类似，参考菜鸟教程
  • $().attr() 获取/设置 HTML 元素的属性，注意与 prop 区别。这个页面有所演示，但是这里暂时不深入探究
  • $().html() 我认为类似原生 DOM 的 innerHTML= 操作，当然还是对所有 jQuery 选择器匹配的元素操作
  • $().text() 对应 innerText=
  • $().remove() 移除页面元素（包括所有子元素）。正常使用不接收参数，但是参数还可以是一个选择器，只删除符合选择器的部分，例如 $("p").remove(".SOMECLASS");
  • $().empty() 保留此元素，但清空其内部所有子元素
  • $().appendTo() 参数还是一个选择器，将主选择器选中的元素移动到 appendTo 参数匹配的元素的内部
  • $().clone() 复制 DOM 元素，通过 $().clone().appendTo() 来进行元素的复制黏贴
  • $().parent() 获取元素的父元素
  • $().children() 获取元素的所有子元素

进阶 css 选择器

以下列出的都是选择器，也就是填入 $() 内部的内容

  • target:nth-child(N), 获取 target 元素的父元素的第 N 个子元素，计数从 1 开始

注意这个真的是非常绕，选中的对象并不是 target 的子元素，而是 target 的同级元素（父元素的子元素）
